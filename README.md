# Sistema de Gestión de Pacientes

## Descripción
Aplicación web para gestión de pacientes con Vue.js que permite CRUD completo con validaciones, búsqueda y notificaciones.

## Requisitos
- Node.js v14+
- npm v6+ o yarn
